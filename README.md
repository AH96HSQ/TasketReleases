## Installation

### Android
1. Download `Tasket.Setup.V1.apk`
2. Enable "Install from unknown sources" if prompted
3. Open the APK and install

### Windows
1. Download `Tasket.Setup.V1.exe`
2. Run the installer
3. Follow the installation wizard

## Requirements

- **Android**: Android 5.0 (API 21) or higher
- **Windows**: Windows 10 or higher

## Privacy

Your data is yours. Tasket only syncs data when you create an account and explicitly enable sync. All data is stored securely.
